# Voxio Tech Website

A modern, animated, and fully responsive static website for Voxio - an online gadget and tech brand.

## Features

- Modern, tech-inspired design with black background and silver/metallic accents
- Fully responsive layout for mobile, tablet, and desktop
- Smooth animations and hover effects
- Product showcase with beautiful product cards
- Dynamic mobile-friendly navigation
- Multiple pages: Home, Info, Specifications, About, Connect
- Ready for GitHub Pages hosting
